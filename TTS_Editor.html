      How to Use and Convert to MP3

      This editor uses your browser's built-in voices (e.g., Microsoft, Google, or Apple voices). To expand voice options, install additional voices via your operating system settings (e.g., Windows Speech Settings, macOS VoiceOver). Convert your TTS config to MP3 with these steps:

      1. Download Config: Click "Download" after previewing or speaking your text.
      2. Choose a Tool: Use a tool like eSpeak or an online service like NaturalReader.
      3. eSpeak Command: Install eSpeak and run (replace config.txt): 
        espeak -f config.txt -w output.mp3
      4. Online Tools: Paste config text into an online TTS service, adjust settings, and download MP3.
      5. Note: Voice availability depends on your device. Check your system settings to add more voices.
      6. Other Options:
        • Balabolka (Windows): Use Balabolka. Paste config text, adjust settings, and export as MP3.
        • ttsMP3.com: Visit ttsMP3.com. Paste text, select voice, and download MP3.
        • IBM Watson TTS: Try IBM Watson. Sign up, input text, and download audio.
        • Google Cloud TTS: Use Google Cloud TTS. Create an account, input text, and generate MP3.
        • Voicemod TTS: Explore Voicemod TTS. Paste text, choose a voice, and download.
